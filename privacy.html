Privacy Policy

Last updated: June 7, 2026

JOSH (the “Application”) is a private, single-user wellness application operated by an individual (the “Operator”) for personal use. This policy describes how the Application handles data.

Who this applies to

The Application is not offered for public registration and has no users other than the Operator. All data in the Application belongs to the Operator and describes the Operator’s own health and activity.

Data the Application handles

The Application stores and processes the Operator’s own personal data, which may include:

  • Body metrics (e.g., weight, body composition)
  • Sleep and recovery metrics (e.g., sleep duration, heart rate variability, readiness)
  • Nutrition and activity data
  • Training and workout records

Connected services

With the Operator’s explicit authorization, the Application may connect to third-party services (such as Oura) to import the Operator’s own data via their official APIs. The Application only requests access to data belonging to the authenticated account, uses it solely to display and analyze that data within the Application, and never sells or shares it with third parties. Authorization can be revoked at any time through the connected service’s account settings.

How data is stored

Data is stored in a secured cloud database. Access to the Application is protected by authentication. Credentials and access tokens for connected services are stored securely and are never exposed publicly.

Data sharing

The Application does not sell, rent, or share personal data with advertisers or unrelated third parties. Data is processed only by the infrastructure providers required to operate the Application (such as hosting and database services).

Data retention and deletion

The Operator may delete data or disconnect any connected service at any time. Disconnecting a service stops further data import and removes its stored access tokens.

Contact

Questions about this policy can be directed to the Operator via the contact email provided during service registration.
